# GHZ-Based Quantum Secret Sharing and Entanglement Analysis Framework

A comprehensive quantum information science project exploring **GHZ-state entanglement**, **multipartite quantum correlations**, **noise resilience**, and **quantum secret sharing protocols**. This repository combines theoretical analysis and practical simulation to investigate how entanglement can be utilized for secure quantum communication and distributed cryptographic applications.

---

## Overview

Greenberger-Horne-Zeilinger (GHZ) states are among the most important multipartite entangled states in quantum information science. They serve as a foundation for:

- Quantum Secret Sharing
- Distributed Quantum Communication
- Multipartite Entanglement Studies
- Quantum Network Protocols
- Tests of Quantum Nonlocality

This project implements and analyzes GHZ states from multiple perspectives, ranging from state preparation and entanglement characterization to security-focused communication protocols.

---

## Key Features

### GHZ State Generation

- Construction of multipartite GHZ states using Qiskit
- Statevector and density matrix analysis
- Measurement statistics and correlation studies

### Entanglement Characterization

- Von Neumann entropy calculations
- Reduced density matrix analysis
- Purity and coherence evaluation
- Multipartite entanglement investigation

### Quantum Nonlocality Verification

- Mermin inequality implementation
- Expectation value calculations
- Analysis of Bell-type violations

### Noise and Robustness Studies

- Depolarizing noise simulations
- Fidelity degradation analysis
- Entanglement resilience benchmarking
- State evolution under realistic imperfections

### Quantum Interference Analysis

- Phase manipulation experiments
- Parity oscillation studies
- Coherence preservation analysis

### GHZ vs W-State Comparison

- Structural entanglement differences
- Reduced state comparisons
- Robustness and information distribution analysis

### Quantum Secret Sharing (HBB99)

- Implementation of the Hillery–Bužek–Berthiaume (HBB99) protocol
- Secret encoding and reconstruction
- Cooperative recovery mechanisms
- GHZ-based distributed secret sharing

###Security Analysis

- Intercept-resend attack simulations
- Insider threat analysis
- Participant collusion scenarios
- Quantum Bit Error Rate (QBER) evaluation
- Security-performance tradeoff studies
